# CRA 환경설정 하기

## axios 추가

> npm i axios

## styled-components 추가

> npm i styled-components

## craco 추가

> npm i --save-dev @craco/craco

craco.config.js 파일 추가 (alias 작성)

> 1. api - axios.js
> 2. components
> 3. pages
> 4. utils
